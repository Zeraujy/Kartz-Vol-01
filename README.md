## 📖 Sobre o projeto

*Kartz* é uma novel original brasileira que mistura **fantasia, mistério e ação**, ambientada em um mundo marcado por conflitos, ruínas e segredos esquecidos pelo tempo.

Este projeto representa o início do **Volume 1**, reunindo escrita, ilustrações e desenvolvimento visual produzidos de forma totalmente independente.

---

## ✨ Apoie o projeto

Cada apoio ajuda diretamente na produção da novel, permitindo o desenvolvimento de novos capítulos, ilustrações e a expansão do universo de *Kartz*.
